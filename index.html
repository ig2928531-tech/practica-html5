Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web.

Comenzar

¿Qué es HTML5?

HTML5 es la quinta versión del lenguaje de marcado HTML, utilizado para estructurar y presentar contenido en la web. Introduce nuevas etiquetas y funcionalidades que permiten crear páginas más interactivas, accesibles y compatibles con distintos dispositivos. Es importante porque facilita el desarrollo de aplicaciones modernas sin necesidad de tantos complementos externos.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

La estructura de un documento HTML5 comienza con >!DOCTYPE html>, que indica al navegador que se trata de HTML5. La etiqueta >html> envuelve todo el contenido. Dentro de >head> se colocan metadatos, el título y enlaces a estilos o scripts. La etiqueta >title> se utiliza para colocar los titulos . En >body> se encuentra el contenido visible para el usuario. La etiqueta >/body> cierra la sección del documento donde se coloca todo el contenido visible para el usuario La etiqueta >/html> indica el final del documento HTML. Esta organización asegura que la página sea clara y funcional.

Etiquetas principales

h1 - h6

Definen títulos y subtítulos, jerarquizando la información.

p

Representa párrafos de texto.

img

Inserta imágenes en la página.

a

Crea enlaces hacia otras páginas o secciones.

button

Genera botones interactivos.

ul / ol / li

Permiten crear listas ordenadas o desordenadas.

HTML5 semántico

header

Es el encabezado de la página o sección.

nav

Es el área de navegación con enlaces.

main

Contenido principal del sitio.

section

Agrupa bloques temáticos.

article

Contenido independiente como artículos o posts.

footer

Pie de página con información adicional.

Formularios en HTML5

Los formularios permiten recopilar información del usuario. Se construyen con la etiqueta

y pueden incluir: label: Etiquetas descriptivas para los campos.
input: Campos de texto, correo, contraseña, etc.
textarea: Áreas de texto más grandes.
select: Menús desplegables.
button: Botones para enviar o resetear.
Son esenciales para registros, búsquedas y comunicación en la web.

Multimedia en HTML5

En esta sección explicamos cómo HTML5 permite trabajar con imágenes, audio y video.

Imágenes

Inserta imágenes con atributos como src y alt.

Audio

Reproduce archivos de sonido directamente en el navegador.

Video

Permite mostrar videos sin necesidad de plugins externos.

Accesibilidad

El atributo alt en imágenes describe el contenido para usuarios con lectores de pantalla.

Conclusión

HTML5 representa un gran avance en el desarrollo web, ya que ofrece herramientas modernas para crear sitios más interactivos, accesibles y compatibles con diferentes dispositivos. Su estructura semántica mejora la organización del contenido y su integración con multimedia y formularios lo convierte en la base de la web actual.
